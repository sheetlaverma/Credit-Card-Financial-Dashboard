# Credit-Card-Financial-Dashboard
**Credit Card Financial Dashboar:**
<br>
**Tool Used:** SQL, Power BI
<br>
**Objective:** To build a weekly dashboard that provides real-time insights into key credit card performance metrics and trends for effective monitoring and analysis.
<br>
**Description:**

- Designed and developed an interactive dashboard using SQL (for data extraction and processing) and Power BI (for visualization).

- Streamlined data analysis to track and monitor performance metrics and trends efficiently.

- Shared actionable insights with stakeholders, supporting data-driven decision-making.
